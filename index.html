Get started in 60 seconds

Get Started in 60 Seconds

1

Set your API keys

Press Ctrl+K and add your OpenAI, Anthropic, Google, or xAI keys. Stored locally in your browser only.

2

Start typing

You'll land in Input Mode. Type your message and press Enter to send.

3

Navigate with j/k

Browse your conversation history with vim-style keys. Press g/G to jump to first/last.

4

Filter with commands

Press Ctrl+D and type queries like t'work' or r30 s>=2 to filter your history.

Security & Privacy

Security & Privacy First

  • Bring Your Own Keys (BYOK) - Use your own API keys from OpenAI, Anthropic, Google, or xAI
  • 100% Local Storage - All conversations stored in your browser (IndexedDB), never sent to any server
  • Your keys, your control - API keys stored in localStorage, sent only to the providers you choose
  • No backend, no tracking - Pure client-side JavaScript, zero data collection
  • Open source - MIT licensed, audit the code yourself on GitHub
